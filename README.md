# Screeps-Genesis
My first screeps repo

## To deploy
- Clone the repo
- Create a file named config.json as follows  
```$json
{
    local: {
        path: "/path/to/screeps/code/folder/default
    }
}
```
- Run ```npm install```
- Run ```grunt --server=local```

## Current features
1. Creeps creation automation
2. Basic harvesting and upgrading
